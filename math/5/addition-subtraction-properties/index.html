Свойства сложения и вычитания

1Свойства сложения

  • От перемены мест слагаемых сумма не меняется. Например, 2 + 3 и 3 + 2 — это одно и то же, оба раза получаем 5.
  • Если слагаемых больше двух, складывать можно в любом удобном порядке. Например, 3 + 4 + 2: сначала 3 + 4 = 7, затем 7 + 2 = 9. А можно и по-другому: 2 + 3 = 5, затем 5 + 4 = 9. Результат одинаковый.
  • Если к числу прибавить ноль, число не меняется. Например, 3 + 0 = 3.

2Свойства вычитания

Если сначала сложить два числа, а потом результат вычесть из третьего, то можно вместо этого вычесть эти два числа по очереди — ответ будет тот же.

•5 − (1 + 2)

1 + 2 = 3, а 5 − 3 = 2. Но можно и по-другому: сначала 5 − 1 = 4, затем 4 − 2 = 2. Ответ тот же самый.

•4 + 3 − 2 — порядок действий можно менять

По порядку: 4 + 3 = 7, 7 − 2 = 5. Но можно сначала 4 − 2 = 2, затем 2 + 3 = 5. Или даже 3 − 2 = 1, затем 1 + 4 = 5. Во всех случаях получаем 5 — главное не потерять знак у каждого числа.

  • Если из числа вычесть ноль, число не меняется. Например, 2 − 0 = 2.
  • Если число вычесть само из себя, получится ноль. Например, 6 − 6 = 0.
